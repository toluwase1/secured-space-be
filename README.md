# Rentals-API

Rentals-API is a `RESTful-API` that allows `developers` to ` connect to data about rental properties`.


## Using Rentals-API

To use Rentals-API, clone this repository and then install the dependencies using:

```
go mod tidy
```

To run, use:


```
make run
```
